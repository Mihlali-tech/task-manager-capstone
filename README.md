# Task Manager Capstone Project

## Overview
A web-based task management system where users can create projects, add tasks, assign priorities and due dates, track progress, and mark tasks as completed. Built with Django and Django REST Framework.

## Features
- User authentication (register/login/logout)
- Create, edit, delete Projects
- Create, edit, delete Tasks within Projects
- Task fields: title, description, status, priority, due date
- Dashboard showing all projects and tasks
- JWT authentication for API endpoints
